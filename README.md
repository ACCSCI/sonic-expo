# sonic

Bilibili 音乐播放器（Expo + React Native）

## 基本信息

- 名称：sonic
- 版本：1.0
- 作者：加速科学(ACCSCI)
- 项目主页：https://github.com/ACCSCI/sonic-expo
- 作者主页：https://github.com/ACCSCI

## 开发

```bash
pnpm install
pnpm start
```

## 构建 Android APK（EAS）

```bash
npx eas-cli login
npx eas-cli build -p android --profile preview
```

## 目录结构

- `app/` 路由页面
- `src/` 业务逻辑
- `logo/` 应用图标
