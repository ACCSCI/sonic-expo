# 项目目标：Bilibili 移动端音乐播放器 (Expo)

## 核心技术栈

- 框架：Expo (React Native)
- 样式：NativeWind (Tailwind CSS)
- 播放：react-native-track-player (支持后台播放和锁屏控制)
- 文件：expo-file-system (处理音频下载和读取)

## 任务清单 (按顺序执行)

### 第一步：环境与基础 UI (Day 1-2)

- 初始化 Expo 项目，配置 NativeWind。
- 搭建底部导航栏：[解析、播放、本地]。
- 实现粘贴板监听功能，自动提取 BV 号。

### 第二步：解析引擎 (Day 3-4)

- 编写解析逻辑：通过 BV 号获取视频基本信息（标题、封面、分P列表）。
- 获取音频流地址：调用 B 站接口，提取音频 URL，并设置正确的 Headers (Referer: <https://www.bilibili.com)。>

### 第三步：播放控制 (Day 5-6)

- 集成播放器，实现流媒体播放。
- 实现后台播放保活、锁屏封面显示、上一曲/下一曲。

### 第四步：下载管理 (Day 7-8)

- 实现下载功能：将音频流保存到手机本地存储。
- 本地列表页：扫描手机中的音频文件，支持离线播放。

### 第五步：异常处理与优化 (Day 9-10)

- 处理 403 报错、网络超时。
- 优化长列表性能。
